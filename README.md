# nodejs-backend-practice

This repository contains my Node.js learning journey and backend development practice. It includes hands-on implementations, examples, and notes covering core Node.js concepts and modern backend technologies.

Topics covered in this repository include:

- Core Node.js concepts and modules
- Functions, timers, streams, and file handling
- HTTP module and server creation
- Routing and status codes
- NPM packages and utilities
- Express.js fundamentals and routing
- Middleware and Morgan
- Static files and view engines (EJS)
- MongoDB Atlas setup
- Mongoose schemas and models
- CRUD operations and REST APIs
- MVC architecture
- Authentication concepts
- Sessions, cookies, and JWT
- Password hashing using bcrypt
- Async/Await and backend workflows

This repository serves as my practice workspace, learning notes, and reference while exploring backend development using Node.js, Express.js, MongoDB, and related technologies.
